# Technology Stack

## Core Configuration & Management

- **GNU Stow**: The primary mechanism for managing symlinks and organizing
  configuration files into modular packages.
- **Homebrew**: Used for system-wide package management and environment
  reproducibility via `Brewfile`.

## Shell Environment

- **Zsh**: The primary interactive shell, enhanced with modular configurations.
- **Antidote**: Fast, zsh-native plugin manager for maintaining shell
  extensions.
- **Starship**: Cross-shell prompt for a consistent and high-performance visual
  experience.
- **Bash**: Used for various automation and maintenance scripts.

## Developer Experience

- **Neovim (LazyVim)**: A high-performance, extensible text editor configured
  with the LazyVim distribution for a full IDE-like experience.
- **Tmux**: Terminal multiplexer for session management and advanced pane
  tiling.
- **Git**: Enhanced with **Delta** for human-readable, side-by-side
  syntax-highlighted diffs.

## macOS Integration & Automation

- **AeroSpace**: Tiling window manager for macOS, optimizing for
  keyboard-driven workspace management.
- **Karabiner-Elements**: Advanced keyboard customization for remapping and
  complex modifications.
- **Raycast**: Extensible application launcher with custom TypeScript-based
  productivity tools.

## Infrastructure & Runtime

- **Docker**: Containerization for localized development and dependency
  management.
- **Kubectl**: CLI for managing Kubernetes clusters, optimized with custom
  Ruby-based wrappers (`kubectl-jget`).
- **Language Runtimes**: Lua (Neovim), TypeScript (Raycast), Ruby, and Perl for
  various system utilities.

## Aesthethics

- **Catppuccin**: A unified, high-quality color scheme applied across all
  supported applications for visual consistency.
